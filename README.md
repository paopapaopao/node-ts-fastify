# node-ts-fastify

A minimal TypeScript + Fastify server.

## Requirement(s)

- Node.js 18+ (recommended: latest LTS)
- pnpm

## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/paopapaopao/node-ts-fastify.git
cd node-ts-fastify
```

2. **Install dependencies**

```bash
pnpm install
```

3. **Start the development server**

```bash
pnpm run dev
```

## Script(s)

| Command             | Description                  |
| ------------------- | ---------------------------- |
| `pnpm run start`    | Start production server      |
| `pnpm run dev`      | Start development server     |
| `pnpm run generate` | Generate migrations          |
| `pnpm run migrate`  | Apply migrations to database |
| `pnpm run studio`   | Open Drizzle studio          |
