# node-ts-fastify

A minimal TypeScript + Fastify server.

## Requirement(s)

- Node.js 18+ (recommended: latest LTS)
- pnpm

## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/paopapaopao/node-ts-fastify.git
cd node-ts-fastify
```

2. **Install dependencies**

```bash
pnpm install
```

3. **Start the development server**

```bash
pnpm run start
```

## Script(s)

| Command          | Description      |
| ---------------- | ---------------- |
| `pnpm run start` | Start dev server |
